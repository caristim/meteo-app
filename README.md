# Meteo App

Aplicación meteorológica simple para Android.

## Funciones

- Registro de mediciones meteorológicas
- Pronóstico estimado para 12 horas
- Gráfica automática de variables
- Guardado local en el navegador
- Compatible con Android

## Tecnologías

- HTML
- CSS
- JavaScript
- Chart.js

## Uso

Abrir index.html en el navegador.
